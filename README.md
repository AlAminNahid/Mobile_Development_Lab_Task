## 📌 Repository Overview

This repository contains a collection of **10 Android lab tasks** completed as part of the Mobile Application Development course. Each lab focuses on different UI components, layouts, and functionalities in Android development using **Kotlin**.

All applications are designed following proper UI/UX practices including:

* Clean and structured layouts
* Responsive design
* Proper spacing and alignment
* Event-driven programming

---

## 🧪 Lab Tasks Summary

### 🔹 Lab Task 01: Student Registration Form App

* Designed a complete student registration form
* Used components like EditText, RadioGroup, CheckBox, Spinner, DatePicker
* Implemented validation and Toast message display
* Reset functionality clears all inputs

---

### 🔹 Lab Task 02: User Login & Profile Card App

* Implemented login system with hardcoded credentials
* Used RelativeLayout for UI positioning
* Displayed profile card after successful login
* Included ProgressBar and logout functionality

---

### 🔹 Lab Task 03: Fitness Tracker Dashboard App

* Built dashboard using ConstraintLayout
* Displayed stats: steps, calories, water intake, workout time
* Used CardViews and ProgressBar
* Dynamic progress update with user input

---

### 🔹 Lab Task 04: Student Grade Report App

* Created grade table using TableLayout
* Dynamic subject addition and GPA calculation
* Implemented grading logic and color highlighting
* Displayed summary and performance metrics

---

### 🔹 Lab Task 05: Contact Book App

* Developed contact list using ListView
* Custom adapter with ViewHolder pattern
* Features: Add, Delete, Search contacts
* Implemented real-time filtering

---

### 🔹 Lab Task 06: News Article Reader App

* Used ScrollView for long content display
* Implemented article sections and navigation buttons
* Added bookmark and share functionality
* Smooth scrolling between sections

---

### 🔹 Lab Task 07: Photo Gallery App

* Built image gallery using GridView
* Implemented fullscreen image view with zoom
* Multi-select mode with delete/share options
* Category-based filtering

---

### 🔹 Lab Task 08: In-App Browser & Learning Portal

* Implemented WebView for browsing
* Custom toolbar with navigation controls
* Address bar with URL validation
* Offline HTML page handling

---

### 🔹 Lab Task 09: E-Commerce Product Listing App

* Used RecyclerView with list & grid modes
* Implemented product filtering and search
* Added cart functionality with badge count
* Swipe-to-delete and drag-to-reorder features

---

### 🔹 Lab Task 10: University Event Management App

* Multi-screen application combining various layouts
* Features: Event listing, details, booking system
* Seat selection grid with dynamic states
* Navigation using Intents and data passing
